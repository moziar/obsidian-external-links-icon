## 插件说明

这是一个用于在链接后面添加图标的 Obsidian 插件。



## 相关工具

你可以在项目目录下的 `skills` 目录中，找到以下 skill：

- defuddle: Use Defuddle CLI to extract clean readable content from web pages.
- json-canvas: define what is json canvas
- obsidian-base: define what is obsidian base
- obsidian-cli: Use the `obsidian` CLI to interact with a running Obsidian instance
- obsidian-markdown: Create and edit valid Obsidian Flavored Markdown. 



你应该先看一遍这些 skill。
